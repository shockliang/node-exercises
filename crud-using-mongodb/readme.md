### Install via brew ###
* `brew install mongodb`
* Making folder for mongodb
    * `sudo mkdir -P /data/db`
* Changing owner for `/data/db` folder
    * `sudo chown -R `id -un` /data/db`
* Run the mongodb daemon
    * `mongod`

* Schema types
    * String
    * Number
    * Date
    * Buffer
        * Stored binary data
    * Boolean
    * ObjectID
        * For assign unique identity.
    * Array
